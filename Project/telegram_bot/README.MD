# feedback_TG_bot

## Описание
Telegram бот для сбора обратной связи от пользователей. Первая версия, без инлайн панели, тестовая

## Функционал
- Сбор отзывов.
- Интеграция с PostgreSQL.
- Использование Docker для контейнеризации.
